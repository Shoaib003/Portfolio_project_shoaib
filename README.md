# Value & Performance Controlling Portfolio Project
Driver-Based Financial Analysis &amp; Rolling Forecast Model for Consulting Departments

# Overview
This project simulates a real-world Value * Performance Controlling setup for a consulting/advisory organization.
It demonstrates how actual performance, driver-based forecasts, and management KPIs can be combined to support data-driven decision-making.

## Business Objective:
- Are we on track financially?
- How do forecasts compare to latest actuals?
- Which business drivers explain performance changes?

# What is in this project included:
### SQL Data Model
- Star schema with fact & dimension tables
- Controller-ready SQL view:
- Monthly performance
- YTD metrics
- Rolling 12-month revenue
- Key financial KPIs

# Python Analysis
- Synthetic but realistic financial & operational data
- Driver-based profitability analysis
- Utilization sensitivity scenarios
- Outputs prepared for Excel & Tableau

# Excel Rolling Forecast
- Driver-based revenue & cost modeling
- scenario planning (Base / Optimistic / Conservative)
- Rolling cost inflation logic
- Variance analysis (Actual vs Forecast)
- Management KPI sheet for executive reporting

# Tools used in this Project
- Python for data simulation
- SQL (SQLite-stlye schema) for Data Modeling
- Excel for Forecasting, dynamic formulas, and scenario
- Google Colab environment
