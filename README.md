# Portfolio_project_shoaib
Driver-Based Financial Analysis &amp; Rolling Forecast Model for Consulting Departments

# Overview
This portfolio project demonstrates an end-to-end controlling solution for a consulting/advisory business.

## This Project covers:
- Data Modeling with SQL (star schema for departments, financials, and operations)
- Driver-based financial analysis with Python (profitablity, utilization, scenario sensitivity)
- Dynamic Excel rolling forecast with scenario planning and variance analysis
- Executive KPI reporting

The goal is to simulate a real-world controlling environment where actual performance, forecast, and key business drivers are monitored and analyzed to support decision-making.
